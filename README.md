# 缘起 / Origin

我是从Python Weekly的第219期邮件推送中发现本书的，打开提供的[链接][book_link]后发现这是免费阅读的书籍。大概看了一下，书也很精简，不足100页，重要的是可以帮助自己发现一些错误。我曾打算翻译一下。原因有三，一是，练习一下自身的英语，二是，虽然英语看起来原滋原味，但母语看起来也更加方便。三是，这本书真的很薄，我喜欢薄的书。

I found this Book in an email sent by Python Weekly. After I opened the [link][book_link], I found the book is free to read and can be download. This book looks short and simple. And the title of this book impiles it will be useful. It can help me to find my mistakes in Python. I plan to translate it in one day. The reasons are very simple. First, practicing my poor english. Second, the English version has the original ideas of the authors, but the Chinese version is more easy to read. Third, this book is really short, translating will not cost too mush time.
